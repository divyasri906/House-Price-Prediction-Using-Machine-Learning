# House Price Prediction using Machine Learning

## Project Overview

This project predicts house prices using a Linear Regression model. The model was trained using housing features such as area, bedrooms, bathrooms, stories, parking, and furnishing status.

## Dataset Information

The dataset contains housing-related features and house prices.

Features:
- Area
- Bedrooms
- Bathrooms
- Stories
- Main Road Access
- Guest Room
- Basement
- Hot Water Heating
- Air Conditioning
- Parking
- Preferred Area
- Furnishing Status

Target Variable:
- Price

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Project Workflow

1. Data Loading
2. Data Exploration

6. Correlation Analysis
7. Data Preprocessing
8. Feature Selection
9. Train-Test Split
10. Linear Regression Model
11. Model Evaluation
12. House Price Prediction

## Model Performance

- Train R² Score: 0.685
- Test R² Score: 0.649
- MAE: 979,679
- RMSE: 1,331,071

## Conclusion

The Linear Regression model achieved good predictive performance and demonstrated effective house price estimation without significant overfitting. Housing area and bathrooms showed a strong influence on house prices.
